## 📖 Sobre o Projeto

O projeto **Agrinho 2026** foi criado para mostrar a importância do equilíbrio entre o agronegócio e a preservação ambiental.

A proposta é conscientizar sobre práticas sustentáveis no campo, incentivando o uso responsável dos recursos naturais e das tecnologias modernas.

---

## 🌿 Temas Apresentados

- Preservação ambiental
- Sustentabilidade no campo
- Economia de água
- Energia renovável
- Tecnologia agrícola
- Produção sustentável
